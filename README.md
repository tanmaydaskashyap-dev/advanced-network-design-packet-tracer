# Advanced Network Design using Cisco Packet Tracer

![Banner](banner3.png) 


## 📌 Project Overview

This project demonstrates a multi-network architecture using:

* Router on a Stick
* HSRP (Hot Standby Router Protocol)
* EIGRP Routing Protocol

## 🧱 Network Components

* 3 Routers (1841)
* Multiple Switches (2950)
* PCs and Server

## 🌐 IP Addressing

* Network 1: 1.0.0.0
* Network 2: 2.0.0.0
* Network 3: 3.0.0.0
* Network 4: 4.0.0.0

## ⚙️ Features Implemented

### 🔹 Router on a Stick

* Inter-VLAN routing configured using subinterfaces

### 🔹 HSRP

* Provides gateway redundancy
* Ensures high availability

### 🔹 EIGRP

* Dynamic routing between routers
* Efficient path selection

## 🧪 Testing

* Successful ping between all networks
* Failover tested using HSRP


## ▶️ How to Run

1. Open the .pkt file in Cisco Packet Tracer
2. Use simulation mode to observe packet flow
3. Test connectivity using ping
